## user api

user

```bash
curl http://127.0.0.1:8080/v1/users/{username}
```

activity

```bash
curl http://127.0.0.1:8080/v1/users/{username}/activity
```

followers

```bash
curl http://127.0.0.1:8080/v1/users/{username}/followers
```

following

```bash
curl http://127.0.0.1:8080/v1/users/{username}/following
```
